# 景点信息查询指南

## 搜索策略

### 发现景点
1. **综合搜索**：`[城市] 必去景点 TOP10`
2. **分类搜索**：
   - 自然风光：`[城市] 自然风景 山水 公园`
   - 人文古迹：`[城市] 古镇 历史 文化遗址`
   - 亲子乐园：`[城市] 亲子游 儿童乐园 动物园`
   - 网红打卡：`[城市] 网红景点 拍照 打卡`

### 景点详情
- 门票信息：`[景点名] 门票价格 2024`
- 开放时间：`[景点名] 开放时间 营业时间`
- 游玩攻略：`[景点名] 游玩攻略 路线推荐`
- 预约方式：`[景点名] 预约 购票渠道`

## ⭐ 景点分级制度（重要！）

**必须将所有景点分为两类：**

### ⭐⭐⭐ 必玩景点（Must See）
- 定义：该目的地的标志性景点，不去等于没来
- 特点：知名度高、评价好、代表性强
- 建议：时间再紧也要安排

### ⭐ 可选景点（Nice to Have）  
- 定义：有特色但非必去，时间充裕可以安排
- 特点：小众、特定人群喜欢、或与必玩景点类型重复
- 建议：根据个人兴趣和时间灵活安排

## 开放时间搜索策略

搜索关键词：
```
[景点名] 开放时间 2024
[景点名] 周一开门吗
[景点名] 闭馆日
[景点名] 预约 购票
[景点名] 最佳游览时间
```

**必须核实的信息：**
1. 具体开放时间（区分淡旺季）
2. 每周闭馆日（很多博物馆周一闭馆）
3. 停止入场时间（通常比关门早30-60分钟）
4. 是否需要预约
5. 节假日是否调整

## 景点筛选逻辑

### 按偏好筛选

**自然风光爱好者**
- 山岳：黄山、泰山、张家界
- 湖泊：西湖、洱海、千岛湖
- 海滨：厦门、三亚、青岛
- 森林公园、湿地公园

**人文历史爱好者**
- 古镇：乌镇、周庄、凤凰
- 博物馆：故宫、国博、省博
- 寺庙：灵隐寺、少林寺
- 历史遗址、名人故居

**亲子家庭**
- 动物园、海洋馆
- 游乐园：迪士尼、欢乐谷
- 科技馆、自然博物馆
- 农场、采摘园

**情侣出游**
- 夜景观赏点
- 浪漫餐厅
- 文艺街区
- 摩天轮、游船

**美食探索**
- 小吃街、夜市
- 老字号餐厅
- 网红店铺
- 当地特色菜

### 按体力筛选

| 体力等级 | 适合景点类型 | 每日景点数 |
|---------|-------------|-----------|
| 轻松 | 平地公园、博物馆、古镇 | 2-3个 |
| 适中 | 小山、景区、主题乐园 | 3-4个 |
| 暴走 | 登山、大型景区、徒步 | 4-5个 |

## 路线优化原则

1. **地理聚合**：同一区域的景点安排在同一天
2. **时间分配**：
   - 早上（8-11点）：需要体力的景点（登山、大景区）
   - 中午（11-14点）：用餐 + 休息或室内景点
   - 下午（14-17点）：室内或休闲景点
   - 傍晚（17-20点）：适合看夕阳/夜景的地方
3. **距离控制**：相邻景点间隔不超过1小时车程
4. **备选方案**：每天准备1个室内备选（应对天气变化）
