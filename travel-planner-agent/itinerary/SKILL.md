## 行程优化原则

### 时间安排
1. **早上**（8-11点）：安排需要体力的景点
2. **中午**（11-14点）：用餐 + 休息
3. **下午**（14-17点）：室内或休闲景点
4. **傍晚**（17-20点）：夜景/夜市

### 路线优化
1. 同一区域的景点放一天
2. 避免来回折返
3. 考虑交通高峰期
4. 留足交通时间

### 节奏控制
1. 每天景点数量：2-4个为宜
2. 预留休息时间
3. 避免行程过满
4. 最后一天安排轻松

### 备选方案
1. 每天准备1个室内备选
2. 标注"如时间充裕可加"的景点
3. 雨天替代方案
