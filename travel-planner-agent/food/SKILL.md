# 餐饮推荐指南

## 核心要求

**必须提供具体、可执行的餐饮信息：**
1. 具体餐厅名称（不能说"找一家餐厅"）
2. 详细地址（精确到街道门牌号）
3. 人均消费（具体金额）
4. 推荐菜品（具体菜名和价格）
5. 营业时间

## 搜索策略

### 发现餐厅
```
[城市] [区域] 美食推荐
[城市] [菜系] 餐厅 人均XX元

```

## 注意事项

1. **搜索最新信息**：餐厅可能歇业或搬迁，搜索确认
2. **标注信息来源**：价格来自大众点评/美团等
3. **考虑位置**：推荐的餐厅应在当天行程路线上
4. **备选方案**：每餐至少提供1-2个备选
5. **特殊需求**：注意用户的忌口、过敏等
