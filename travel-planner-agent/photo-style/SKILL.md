## 📷 拍照技巧

### 构图建议
1. **三分法**：人物放在画面1/3处
2. **对称构图**：适合建筑、走廊
3. **框架构图**：利用门窗、树枝
4. **引导线**：道路、栏杆延伸

### 姿势参考
| 场景 | 推荐姿势 |
|------|---------|
| 街道 | 走路抓拍、回眸、假装打电话 |
| 咖啡厅 | 看窗外、喝咖啡、看书/手机 |
| 自然风光 | 背影、张开双臂、坐着远眺 |
| 建筑前 | 站立仰拍、台阶坐姿、倚靠 |

### 最佳拍摄时间
| 时间 | 光线特点 | 适合拍摄 |
|------|---------|---------|
| 日出后1小时 | 金色柔光 | 户外人像 |
| 上午10点前 | 柔和自然 | 街拍、建筑 |
| 下午3-5点 | 温暖斜阳 | 户外人像 |
| 日落前1小时 | 黄金时刻 | 剪影、逆光 |
| 蓝调时刻 | 天空深蓝 | 城市夜景 |

## 热门目的地速查

### 首尔
- **风格**：韩系小清新、都市时尚
- **推荐穿搭**：oversized西装、针织衫、百褶裙、小白鞋
- **配色**：奶白、浅蓝、淡粉、米色
- **出片地**：北村韩屋、益善洞、梨泰院、汉江公园

### 东京
- **风格**：日系文艺、潮流街头
- **推荐穿搭**：衬衫、阔腿裤、帆布鞋、贝雷帽
- **配色**：黑白灰、牛仔蓝、墨绿
- **出片地**：浅草寺、涩谷、原宿、�的根

### 巴黎
- **风格**：法式优雅、复古浪漫
- **推荐穿搭**：风衣、贝雷帽、条纹衫、丝巾
- **配色**：黑白红、驼色、酒红
- **出片地**：埃菲尔铁塔、卢浮宫、蒙马特、塞纳河

### 泰国
- **风格**：热带度假、波西米亚
- **推荐穿搭**：吊带裙、碎花裙、草帽、人字拖
- **配色**：白色、橙色、黄色、印花
- **出片地**：大皇宫、清迈古城、海滩、寺庙
