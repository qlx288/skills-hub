# MBTI 个性化旅行定制

## 触发场景
用户提到 MBTI、性格类型、"I人/E人" 等

## 流程

1. 根据用户提供的信息程度，灵活处理：
   - 完整四字母（如"INFP"）→ 直接搜索
   - 部分信息（如"I人"、"IN型"）→ 基于已知维度定制，不追问
   - 只说"内向/外向" → 当作 I/E 处理，不追问

2. 搜索对应的旅行偏好：
```
   完整类型：WebSearch: "INFP" 旅行风格 推荐
   部分信息：WebSearch: "内向型人格" 旅行风格 推荐
```

3. 提取关键维度应用到行程：
   - I人：松散节奏、独处时间、避开人多的地方
   - E人：丰富行程、社交机会、热闹的地方
   - N型：深度体验、文化探索、小众路线
   - S型：感官体验、实际活动、热门景点
   - F型：氛围、情感体验、文艺场所
   - T型：实用、高效、知识性景点
   - J型：详细计划、按时执行
   - P型：灵活弹性、随心所欲

4. 将偏好传递给其他子 skill 进行定制

## 输出格式

💡 根据你的 MBTI（[类型]），行程已定制：
✨ 行程特点：[3-5条个性化特点]
📍 特别推荐：[符合性格的景点/体验]
