# 天气查询指南

## 数据源

使用 wttr.in 服务获取天气：
- 基础格式：`https://wttr.in/城市名?lang=zh&format=3`
- 详细格式：`https://wttr.in/城市名?lang=zh`
- 多日预报：`https://wttr.in/城市名?lang=zh&format=v2`

## 查询方法

使用 WebFetch 工具访问天气服务：

```
WebFetch https://wttr.in/杭州?lang=zh
WebFetch https://wttr.in/Hangzhou?lang=zh&format=3
```

对于中文城市名，可以使用拼音或中文：
- 北京：Beijing 或 北京
- 上海：Shanghai 或 上海
- 杭州：Hangzhou 或 杭州

## 天气对行程的影响建议

| 天气情况 | 建议 |
|---------|------|
| 晴天 | 适合户外景点，注意防晒，建议早出晚归避开中午高温 |
| 多云 | 最佳出游天气，适合各类活动 |
| 小雨 | 准备雨具，可安排室内景点如博物馆、商场 |
| 大雨/暴雨 | 建议室内活动，避免山区和水边活动 |
| 高温(>35°C) | 避开中午户外，建议早晚出行，多补水 |
| 低温(<5°C) | 注意保暖，减少户外时间，选择有暖气的室内景点 |
| 雾霾 | 减少户外活动，佩戴口罩，选择室内景点 |

## 根据天气的打包建议

### 晴天/高温
- 防晒霜（SPF30+）
- 太阳镜
- 遮阳帽
- 薄外套（空调房用）
- 补水喷雾
- 便携风扇

### 雨天
- 雨伞（折叠伞便携）
- 雨衣（户外活动推荐）
- 防水鞋或鞋套
- 防水袋（保护电子设备）

### 低温
- 厚外套/羽绒服
- 围巾、手套、帽子
- 暖宝宝
- 保温杯

## 天气信息输出模板

```
🌤️ [目的地] 天气预报

📅 [日期]
- 天气：[晴/多云/雨等]
- 温度：[最低温]°C ~ [最高温]°C
- 风力：[风向] [风力等级]
- 降水概率：[百分比]

📝 出行建议：
[根据天气给出的具体建议]

🎒 建议携带：
[根据天气的打包清单]
```
