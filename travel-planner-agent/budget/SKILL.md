# 预算分配指南

## ⚠️ 核心原则

**实际费用必须贴近用户预算！**

- ❌ 错误：用户预算8000，实际规划只花4000
- ✅ 正确：用户预算8000，实际规划7500-8500

**预算利用率目标：90%-105%**

如果用户预算明显过高或过低，应该：
1. 告知用户实际情况
2. 建议调整预算或行程
3. 提供不同档次方案

### 计算方法

```
1. 计算日均预算 = (总预算 - 往返交通) ÷ 天数 ÷ 人数
2. 确定档次
3. 按档次匹配住宿、餐饮、景点
4. 验算总费用是否在预算的 90%-105%
```

### 预算不够时的处理

如果按用户要求无法在预算内完成：
1. 明确告知"您的预算可能略紧张"
2. 给出两个方案：
   - 方案A：维持天数，降低档次
   - 方案B：维持档次，减少天数
3. 让用户选择

## 费用计算公式

### 总体计算

```
总预算 = 大交通费 + (住宿费 × 晚数) + (餐饮费 × 天数) + 门票费 + 本地交通 + 备用金

人均预算 = 总预算 ÷ 人数
```

### 分项估算

**交通费**
```
大交通 = 往返票价 × 人数
本地交通 = 日均交通费 × 天数 × 人数
日均交通费参考：30-80元/人/天
```

**住宿费**
```
住宿费 = 房价 × 晚数 × 房间数
注意：晚数 = 天数 - 1
```

**餐饮费**
```
餐饮费 = (早餐 + 午餐 + 晚餐 + 小吃) × 天数 × 人数
参考标准：
- 经济：50-80元/人/天
- 舒适：100-150元/人/天
- 品质：200-300元/人/天
```

**门票费**
```
门票费 = Σ(各景点门票) × 人数
注意：学生证、老人证可享优惠
```

## 省钱技巧

### 交通省钱
1. ✈️ 飞机：提前2-4周订票，关注特价
2. 🚄 高铁：购买往返票，二等座性价比高
3. 🚗 自驾：3人以上分摊更划算
4. 🚕 打车：非高峰期、拼车、顺风车

### 住宿省钱
1. 📅 平日入住比周末便宜20-30%
2. 🏷️ 提前预订有优惠
3. 📱 比价多个平台
4. 🎁 会员积分可抵扣

### 门票省钱
1. 🆓 关注免费开放日
2. 🎫 购买套票/联票
3. 🎓 带好学生证、老年证
4. 📱 网上预订比现场便宜

### 餐饮省钱
1. 🍜 吃当地人去的店，避开景区
2. ☀️ 午餐吃好，晚餐简单
3. 🏪 便利店、超市补充零食
4. 💧 自带水杯，景区水贵

## 预算调整建议

### 预算不足时
1. 缩短行程天数
2. 降低住宿档次
3. 减少收费景点
4. 选择公共交通

### 预算充足时
1. 升级住宿体验
2. 增加特色活动（演出、体验）
3. 尝试更多美食
4. 购买纪念品
