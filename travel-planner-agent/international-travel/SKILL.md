# 出境旅游指南

## 核心要求

**出境游的所有信息必须包含以下内容：**

### 1. 多语言名称（必须！）

所有景点、酒店、餐厅、菜品都要提供三语名称，方便复制搜索：

```
🏛️ 景福宫
- 英文：Gyeongbokgung Palace
- 韩文：경복궁
- 地址：161 Sajik-ro, Jongno-gu, Seoul
- 韩文地址：서울특별시 종로구 사직로 161
```

```
🍜 参鸡汤
- 英文：Samgyetang / Ginseng Chicken Soup
- 韩文：삼계탕
```

```
🏨 首尔新罗酒店
- 英文：The Shilla Seoul
- 韩文：서울신라호텔
- 地址：249 Dongho-ro, Jung-gu, Seoul
- 韩文地址：서울특별시 중구 동호로 249
```

### 2. 双币种价格（必须！）

所有价格必须同时显示当地货币和人民币：

```
💰 门票：₩3,000（约 ¥16）
💰 人均：₩25,000（约 ¥135）
💰 房价：€150/晚（约 ¥1,170/晚）
💰 租车：$65/天（约 ¥470/天）
```

- **中国驾照**：[是否认可]
- **驾照翻译件**：[是否需要，如何办理]

### 租车平台推荐

| 平台 | 网址 | 特点 | 价格参考 |

| Rentalcars | https://www.rentalcars.com | 比价平台，选择多 | $XX/天起 |

| Budget | https://www.budget.com | 性价比高 | $XX/天起 |

### 租车流程

1. **提前预订**（建议出发前1-2周）
   - 选择取还车地点和时间
   - 选择车型
   - 选择保险方案

2. **取车时需要**
   - 护照原件
   - 国际驾照或驾照翻译件
   - 预订确认单
   - 信用卡（用于押金冻结）

3. **还车检查**
   - 加满油（或选择满油还车方案）
   - 检查车辆外观
   - 保留还车凭证

### 保险选择

| 保险类型 | 说明 | 是否必要 |
|---------|------|---------|
| 基础险（CDW/LDW）| 碰撞损失险 | ✅ 必须 |
| 盗抢险（TP）| 车辆被盗 | ✅ 建议 |
| 第三方责任险 | 伤害他人/财产 | ✅ 必须 |
| 超级全险 | 0免赔，全覆盖 | 💡 强烈推荐 |
| 道路救援 | 抛锚、爆胎等 | 💡 建议 |

**推荐**：在租租车购买"超级全险"，比在租车公司买便宜，理赔也方便

## 旅游保险

### 保险信息模板

```markdown
## 🛡️ 旅游保险

### 是否必须购买
- **签证要求**：[某些国家如申根要求必须有保险]
- **最低保额要求**：[如申根要求医疗保额≥3万欧元]

### 保险类型

| 类型 | 保障内容 | 建议 |
|------|---------|------|
| 医疗保障 | 意外受伤、突发疾病 | ✅ 必须，保额建议≥30万 |
| 意外身故/伤残 | 身故或残疾赔付 | ✅ 建议 |
| 航班延误 | 延误超X小时赔付 | 💡 推荐 |
| 行李丢失/延误 | 行李损失赔付 | 💡 推荐 |
| 个人财物 | 被盗、抢劫 | 可选 |
| 旅程取消 | 因故取消行程 | 可选 |
| 紧急救援 | 医疗转运、翻译 | ✅ 重要 |

### 购买渠道

**方式一：支付宝/微信**
- 支付宝：搜索"境外旅游险"
- 保险公司：众安、平安、太平洋等
- 优点：方便快捷，即时生效
- 价格：约¥XX-XXX（根据天数和保额）

**方式二：专业平台**
| 平台 | 网址 | 特点 |
|------|------|------|
| 开心保 | https://www.kaixinbao.com | 比价方便 |
| 慧择网 | https://www.huize.com | 产品全 |
| 美亚保险 | https://www.aig.com.cn | 国际品牌，救援网络强 |
| 安联保险 | https://www.allianz.cn | 欧洲游推荐 |

**方式三：信用卡附赠**
- 部分高端信用卡含旅行保险
- 需用该卡购买机票才能激活
- 保额可能不够，建议补充购买

### 申根签证保险要求（欧洲游必看）

如果去申根国家（法国、德国、意大利、西班牙等），签证要求：
- ✅ 医疗保额 ≥ 30,000 欧元（约23万人民币）
- ✅ 包含紧急医疗遣返
- ✅ 保险期限覆盖整个行程
- ✅ 在所有申根国家有效

**推荐产品**：
- 美亚"万国游踪"
- 安联"申根签证保险"
- 平安"欧洲签证保险"

### 购买建议

```
行程：[X天]
目的地：[国家]
建议保额：医疗 ≥ ¥XXX万，意外 ≥ ¥XX万
预计保费：约¥XX-XXX

推荐产品：
1. [产品名] - ¥XX - [平台]
2. [产品名] - ¥XX - [平台]
```

### ⚠️ 理赔注意
- 保留所有医疗单据原件
- 第一时间联系保险公司报案
- 航班延误需要航空公司证明
- 财物丢失需要报警证明
```
