# 目的地探索指南

## 使用场景

当用户说出以下类似的话时，触发此 Skill：
- "我想去XXX玩"（没有具体天数/预算）
- "XXX怎么样？值得去吗？"
- "XXX有什么好玩的？"
- "去XXX要准备多少钱？"
- "XXX玩几天合适？"

## 搜索策略

### 基础信息搜索
```
[目的地] 旅游攻略 介绍
[目的地] 是个什么样的城市
```

## 工作流程

1. **用户说"我想去XXX玩"**
2. **触发目的地探索**，输出完整介绍
3. **引导用户明确需求**：
   - 玩几天？
   - 预算多少？
   - 偏好什么？
4. **用户回答后**，再进入详细行程规划
