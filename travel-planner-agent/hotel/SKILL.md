# 酒店查询指南

## 搜索策略

### 发现酒店
1. **区域搜索**：`[城市] [景区/商圈名] 附近酒店推荐`
2. **价格搜索**：`[城市] [价格区间] 酒店 性价比`
3. **类型搜索**：`[城市] 民宿/青旅/亲子酒店/五星酒店`
4. **评价搜索**：`[城市] 酒店推荐 高评分`

### 酒店详情
- 价格信息：`[酒店名] 价格 房型`
- 位置交通：`[酒店名] 位置 交通 地铁`
- 评价口碑：`[酒店名] 评价 怎么样`

## 酒店档次参考

| 档次 | 价格区间(每晚) | 特点 | 适合人群 |
|------|---------------|------|---------|
| 经济型 | 100-200元 | 干净整洁，设施基础 | 预算有限、短住 |
| 舒适型 | 200-400元 | 位置好，服务周到 | 普通出游首选 |
| 高档型 | 400-800元 | 设施完善，体验好 | 品质出行、商务 |
| 豪华型 | 800-1500元 | 顶级服务，景观房 | 度假、纪念日 |
| 奢华型 | 1500元以上 | 极致体验，专属服务 | 蜜月、特殊场合 |

## 住宿类型对比

| 类型 | 优点 | 缺点 | 适合场景 |
|------|------|------|---------|
| 连锁酒店 | 标准化、稳定 | 缺少特色 | 商务、短住 |
| 精品酒店 | 设计感、独特 | 价格偏高 | 情侣、拍照 |
| 民宿 | 有特色、当地体验 | 服务不稳定 | 深度游、体验 |
| 青年旅舍 | 便宜、社交 | 隐私性差 | 独行、穷游 |
| 度假酒店 | 设施全、一站式 | 远离市区 | 亲子、度假 |
| 公寓式 | 有厨房、宽敞 | 服务有限 | 长住、家庭 |

## 住宿选择因素

### 核心考虑

1. **位置**（权重最高）
   - 靠近主要景点？
   - 靠近地铁/公交站？
   - 周边餐饮购物？
   
2. **价格**
   - 符合预算？
   - 含早餐？
   - 有隐藏费用？

3. **评分**
   - 4.5分以上优先
   - 关注差评内容
   - 看近期评价

4. **设施**
   - 免费WiFi
   - 停车场
   - 行李寄存
   - 洗衣服务

### 特殊需求

| 人群 | 重点关注 |
|------|---------|
| 亲子 | 儿童设施、加床、安全 |
| 情侣 | 浪漫氛围、景观房、浴缸 |
| 老人 | 无障碍、电梯、安静 |
| 商务 | WiFi、办公桌、早餐 |

## 区域选择建议

### 通用原则
- **第一次去**：住市中心或主景区附近
- **深度游**：可以换不同区域体验
- **亲子游**：选有游乐设施的酒店
- **预算有限**：选地铁沿线经济型

### 热门城市住宿区域

**北京**
- 前门/王府井：近故宫天安门，购物方便
- 三里屯/国贸：现代繁华，酒吧美食
- 后海/南锣：胡同文化，文艺氛围

**上海**
- 外滩/南京路：经典地标，夜景绝佳
- 静安/淮海路：购物天堂，交通便利
- 迪士尼周边：亲子首选

**杭州**
- 西湖边：风景最佳，价格较高
- 河坊街：古色古香，美食多
- 武林广场：商圈中心，交通便利

## 省钱技巧

1. **时间选择**
   - 提前2-4周预订
   - 平日比周末便宜20-30%
   - 避开节假日和旺季

2. **平台比价**
   - 携程、美团、飞猪、去哪儿
   - 官网可能有会员价
   - 关注平台优惠券

3. **房型选择**
   - 无窗房便宜但体验差
   - 大床/双床价格可能不同
   - 看清是否含早餐

4. **其他技巧**
   - 酒店会员积分可抵扣
   - 连住多晚可议价
   - 非核心区域更划算

## ⚠️ 酒店选择核心原则

### 1. 位置要求（最重要）

**酒店必须靠近当天主要景点，距离控制在 3 公里以内！**

| 情况 | 处理方式 |
|------|---------|
| 景点集中在一个区域 | 选该区域的酒店，全程不换 |
| 景点分散在不同区域 | 可以换酒店，每晚住最近的区域 |
| 跨城市行程 | 每个城市单独选酒店 |

**距离标注要求：**
- 必须标注酒店到每个主要景点的距离
- 距离超过3公里需要特别说明原因
- 标注步行/地铁/打车时间

### 2. 档次要求

**无特殊要求时，默认选择中等档次（舒适型）**

| 档次 | 价格区间 | 代表品牌 |
|------|---------|---------|
| 经济型 | 150-250元 | 如家、汉庭、7天 |
| **舒适型（默认）** | **250-500元** | **全季、亚朵、维也纳、智选假日** |
| 高档型 | 500-1000元 | 希尔顿、万豪、洲际旗下 |

## 多日行程酒店规划

### 景点集中型（推荐同一酒店）
```
Day1-3 住同一酒店
理由：所有景点都在3公里内，不用换酒店更方便
```

### 景点分散型（建议换酒店）
```
Day1-2：住[区域A]的[酒店名]
  - 靠近景点：[景点1]、[景点2]
  
Day3-4：住[区域B]的[酒店名]  
  - 靠近景点：[景点3]、[景点4]

换酒店说明：两个区域相距XX公里，换酒店可节省每天XX分钟通勤
```

## 住宿费用计算

```
住宿总费用 = 房价 × 晚数 × 房间数

示例：
- 房价：¥300/晚
- 入住：3晚
- 房间：1间
- 总计：¥300 × 3 = ¥900
```
